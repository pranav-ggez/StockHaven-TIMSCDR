StockHaven

StockHaven is a web-based stock and cryptocurrency analysis platform that provides real-time market insights through interactive visualizations and comparative analytics. It brings together multiple financial data sources into a single, intuitive interface, allowing users to analyze price movements, compare assets, and explore market trends efficiently.

Overview

The application fetches live and historical market data, processes it on the backend, and presents it through structured tables and candlestick charts. It supports localized currency conversion and includes optional assisted insights to help users interpret short-term trends. StockHaven is designed with simplicity and clarity in mind, making financial data easier to understand without relying on multiple platforms.

Features
-Real-time stock and cryptocurrency data
-Interactive candlestick chart visualization
-Multi-asset comparison table
-INR-based currency conversion
-Portfolio simulation
-Watchlist support
-Assisted trend insights

Tech Stack

Backend: Python, Flask
Frontend: HTML, CSS, JavaScript, Jinja2
Data Sources: Yahoo Finance (yFinance), Binance API
Visualization: Plotly
AI Integration: Google Gemini (optional)
Tools: Git, GitHub, dotenv, Gunicorn

Note!
This project is intended for educational and analytical purposes. It does not support live trading and should not be considered financial advice.
